# Minggu 7

Tanggal : 27 Juni 2026

## Topik

Dynamic Analysis Menggunakan x64dbg

## Yang Dipelajari

Pada minggu ini saya mempelajari konsep dynamic analysis menggunakan x64dbg. Berbeda dengan static analysis, dynamic analysis dilakukan ketika program sedang berjalan sehingga saya dapat mengamati perubahan register, memory, maupun alur eksekusi secara langsung.

Saya juga mempelajari penggunaan breakpoint untuk menghentikan program pada bagian tertentu sehingga proses analisis menjadi lebih mudah.

Materi ini membuat saya memahami bahwa static analysis dan dynamic analysis saling melengkapi dalam proses reverse engineering.

## Yang Masih Membingungkan

Saya masih perlu latihan menentukan lokasi breakpoint yang tepat agar proses debugging menjadi lebih efisien.

## Rencana Minggu Berikutnya

Melakukan analisis sederhana terhadap beberapa CrackMe sebagai latihan.
