# Minggu 1

Tanggal : 10 Maret 2026

## Topik

Pengenalan Reverse Engineering

## Yang Dipelajari

Pada minggu pertama saya mempelajari pengertian Reverse Engineering beserta tujuan utamanya. Reverse Engineering merupakan proses memahami cara kerja sebuah perangkat lunak melalui analisis terhadap executable tanpa memiliki source code asli.

Saya juga mempelajari berbagai bidang yang menggunakan Reverse Engineering seperti malware analysis, software debugging, digital forensics, vulnerability research, serta crackme challenge.

Selain itu saya mengetahui bahwa Reverse Engineering bukan hanya digunakan oleh hacker, tetapi juga oleh software engineer, security researcher, dan analis malware.

## Yang Masih Membingungkan

Saya masih belum memahami bagaimana compiler menerjemahkan source code menjadi bahasa mesin serta bagaimana proses decompiler dapat menghasilkan kode yang menyerupai source code asli.

## Rencana Minggu Berikutnya

Saya ingin mempelajari struktur file executable Windows terutama format Portable Executable (PE).
