# Minggu 4

Tanggal : 26 Juni 2026

## Topik

Strings Analysis

## Yang Dipelajari

Pada minggu ini saya mempelajari teknik analisis menggunakan Strings. Saya mengetahui bahwa banyak executable masih menyimpan informasi dalam bentuk string seperti pesan error, nama file, registry, URL, maupun password yang belum dienkripsi.

Saya mencoba menggunakan Sysinternals Strings untuk melihat isi executable. Dari hasil tersebut saya memahami bahwa informasi sederhana dapat membantu mempercepat proses reverse engineering sebelum membuka executable menggunakan decompiler.

Saya juga belajar bahwa tidak semua string memiliki arti penting sehingga diperlukan proses penyaringan agar fokus terhadap informasi yang relevan.

## Yang Masih Membingungkan

Saya masih belum mengetahui kapan sebuah string berasal dari compiler dan kapan berasal dari logika program.

## Rencana Minggu Berikutnya

Belajar membaca Import Table serta API Windows yang digunakan executable.
