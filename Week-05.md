# Minggu 5

Tanggal : 26 Juni 2026

## Topik

Import Table

## Yang Dipelajari

Saya mempelajari bagaimana executable memanggil fungsi-fungsi yang berasal dari Windows API. Informasi tersebut dapat dilihat melalui Import Table yang berisi daftar library serta fungsi yang digunakan program.

Saya mengetahui bahwa hanya dengan melihat API yang digunakan, seorang analis dapat memperkirakan perilaku executable. Contohnya penggunaan fungsi File API menunjukkan adanya aktivitas terhadap file, sedangkan Registry API menunjukkan adanya akses terhadap registry Windows.

Materi ini membantu saya memahami pentingnya Import Table dalam proses static analysis.

## Yang Masih Membingungkan

Saya masih perlu mempelajari fungsi masing-masing Windows API secara lebih mendalam.
